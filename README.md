# Template

React + Vite template with Tailwind CSS, Radix UI components, and Zustand state management.

## Setup

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint

## Stack

- React 18 + JavaScript
- Vite 6
- Tailwind CSS 4
- Shadcn
- Zustand
- Lucide React icons
